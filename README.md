## üöÄ How to Use

### Setup
1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. No server or installation required - runs completely offline

### Taking the Exam
1. **Register**: Enter your full name and matric number
2. **Start Exam**: Click "Start Exam" button
3. **Answer Questions**: 
   - Click on any option to select your answer
   - Use Previous/Next buttons to navigate
   - Click on question numbers to jump to specific questions
4. **Monitor Time**: Watch the timer in the top-right corner
   - Green: Normal time
   - Orange: 5 minutes remaining (warning)
   - Red: 1 minute remaining (danger)
5. **Submit**: Click "Submit Exam" when done or wait for auto-submit

### After Submission
- View your score percentage and grade
- See correct vs wrong answers breakdown
- Review performance by topic
- Check detailed answer corrections
- Option to retake exam with new questions
